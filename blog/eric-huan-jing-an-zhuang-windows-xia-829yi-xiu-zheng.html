Eric 環境安裝-Windows (下) 8/29已修正

Date Tue 29 August 2017 By 40423152 Tags 暑期學習

介紹安裝方法-2

下載完QT安裝客戶端之後

雙擊開啟 進入安裝流程

按下NEXT之後 登入帳號

帳號可在Qt申請

選擇安裝位置並選擇需要安裝的項目

預設最新版本與工具包即可

按下同意即可繼續安裝

安裝過程 漫長的等待

完成

接下來打開命令提式字元

先切換至C槽

cd C:\

接著使用pip進行安裝PyQt以及QScintilla

兩個打在一起他會自動依序安裝

pip install pyqt QScintilla

若是電腦內原本就有安裝PyQt以及QScintilla

使用-U(要大寫)來進行更新

pip install pyqt5 QScintilla -U

安裝完成後去Eric的官方網站下載Eric的檔案

下載完後進行解壓縮並將命令提示字元的位置轉移到其路徑

以下用我這邊來說明

由於我讓下載目錄預設在Downloads裡面故路徑為這樣

C:\Users\kmol\Downloads\eric6-17.08 (1)\eric6-17.08

這樣就完成切換

並且由於Eric的安裝檔為PY

故必須用Python來進行安裝

python install.py

安裝完成後於命令提示字元輸入eric6即可開啟
